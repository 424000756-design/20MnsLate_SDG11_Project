# 20MnsLate_SDG11_Project
Final Project for IT Elective (Aguirre,Vera,Iloseo)

  A VB.NET Windows Forms application developed to support United Nations SDG 11: Sustainable Cities and Communities by improving the management and monitoring of building inspection records.
The system uses Microsoft SQL Server with ADO.NET for database connectivity and includes features such as:

-Role-based login system
-CRUD operations
-Building inspection record management
-Report generation
-Input validation and error handling

  The application helps organize and monitor building inspection data through a centralized database system, making record management more efficient and reliable compared to manual methods. It also demonstrates the integration of VB.NET, SQL Server, and Windows Forms Application Development concepts.
